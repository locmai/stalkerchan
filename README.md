# Stalker-chan

A stalking bot built for learning Rust purpose.

## Getting started

To run the Stalker-chan API server, first copy the `config.yaml.sample` into `config.yaml`. Then:

To run with Cargo:

```
cargo run
```

To build and run with Docker

```
docker-compsoe up --build
```
